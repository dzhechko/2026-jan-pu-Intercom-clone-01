# Руководство по интерфейсу админ-панели

## Навигация

Боковая панель (sidebar) содержит основные разделы:

| Пункт меню | Путь | Описание |
|------------|------|----------|
| Дашборд | `/` | Основные метрики и графики |
| Разговоры | `/conversations` | Список всех диалогов |
| Лиды | `/leads` | Квалифицированные потенциальные клиенты |
| ROI-аналитика | `/roi` | Расчет экономии и эффективности |

---

## Страница входа (Login)

При первом открытии панели отображается форма авторизации.

### Поля формы

- **Email** -- адрес электронной почты администратора.
- **Пароль** -- пароль учетной записи.

### Dev-режим

В режиме разработки используйте:
- Email: `admin@cloud.ru`
- Пароль: `admin123admin`

После успешного входа JWT-токен сохраняется в localStorage и автоматически прикрепляется ко всем запросам к API. Время жизни токена -- 60 минут.

## Дашборд (Dashboard)

Главная страница с обзором ключевых показателей за выбранный период.

### Выбор периода

Переключатель в верхней части страницы:
- **Сегодня** -- метрики за текущий день.
- **7 дней** -- метрики за последнюю неделю (по умолчанию).
- **30 дней** -- метрики за последний месяц.

### Карточки метрик

Четыре карточки в верхней части:

| Карточка | Описание |
|----------|----------|
| Всего консультаций | Общее количество обработанных диалогов за период |
| Сгенерировано лидов | Количество квалифицированных потенциальных клиентов |
| Среднее время ответа | Средняя скорость ответа агентов (мс) |
| Доля эскалаций | Процент диалогов, переданных живому специалисту |

### График дневного тренда

Линейный график (Recharts) с двумя линиями:
- **Синяя линия** -- количество консультаций по дням.
- **Зеленая линия** -- количество сгенерированных лидов по дням.

Ось X -- дата, ось Y -- количество. При наведении на точку отображается всплывающая подсказка с точными значениями.

### Столбчатая диаграмма "Топ интентов"

Горизонтальная диаграмма, показывающая 5 наиболее частых намерений пользователей:
- Название интента.
- Количество обращений.
- Процент от общего числа.

Примеры интентов: architecture, tco_calculation, compliance_152fz, vmware_migration, ai_factory.

## Страница разговоров (Conversations)

Путь: `/conversations`

### Таблица разговоров

Пагинированная таблица со следующими столбцами:

| Столбец | Описание |
|---------|----------|
| ID | UUID разговора (сокращенный) |
| Канал | Иконка и название: Telegram, Web, API |
| Статус | Бейдж со статусом: active (синий), completed (зеленый), escalated (красный) |
| Интент | Определенное намерение пользователя |
| Дата создания | Время начала диалога |

### Пагинация

- По умолчанию отображается 20 записей на странице.
- Навигация: кнопки "Назад" / "Вперед" и номера страниц.
- Общее количество записей отображается рядом с пагинацией.

### Статусы

- **active** (синий бейдж) -- диалог в процессе, агент активно отвечает.
- **completed** (зеленый бейдж) -- диалог успешно завершен.
- **escalated** (красный бейдж) -- передан живому специалисту.

## Страница лидов (Leads)

Путь: `/leads`

### Таблица лидов

| Столбец | Описание |
|---------|----------|
| Контакт | Имя, email, телефон, компания (из JSONB-поля `contact`) |
| Квалификация | Бейдж: cold (серый), warm (желтый), qualified (синий), hot (красный) |
| Интент | Основное намерение клиента |
| Сумма сделки | Оценочная стоимость проекта в рублях |
| Дата | Дата создания лида |

### Квалификационные бейджи

| Бейдж | Цвет | Значение |
|-------|------|----------|
| cold | Серый | Первичный контакт, намерение не определено |
| warm | Желтый | Проявлен интерес, задает технические вопросы |
| qualified | Синий | Подтвержден бюджет и потребность |
| hot | Красный | Готов к сделке, требуется срочный контакт |

### Пагинация

Аналогична странице разговоров: 20 записей на странице.

## Страница ROI-аналитики (ROI Analytics)

Путь: `/roi`

### Выбор периода

Переключатель: 7 дней, 30 дней (по умолчанию), 90 дней.

### Блок "Экономия ресурсов SA"

| Метрика | Описание |
|---------|----------|
| Часы SA сэкономлены | Количество часов работы solutions architect, которые заменил AI |
| Стоимость экономии | Денежный эквивалент (ставка SA: 5000 руб/час) |
| AI обработал | Количество консультаций, полностью обработанных AI |
| Эскалировано на SA | Количество консультаций, переданных специалисту |

Расчет: каждая AI-консультация экономит 45 минут времени SA (отраслевой бенчмарк).

### Блок "Pipeline и конверсия"

| Метрика | Описание |
|---------|----------|
| Всего лидов | Общее количество сгенерированных лидов |
| Квалифицированных лидов | Лиды со статусом hot или qualified |
| Стоимость pipeline | Суммарная оценочная стоимость квалифицированных сделок |
| Средний чек | Средняя оценочная стоимость сделки |
| Конверсия | Отношение лидов к общему числу консультаций |

### Круговая диаграмма "AI vs SA"

Показывает соотношение:
- Доля консультаций, обработанных AI.
- Доля консультаций, эскалированных на живого специалиста.
