Cloud.ru AI Consultant Widget

This page demonstrates the embeddable chat widget. Look for the blue chat bubble in the bottom-right corner.

The widget will show a proactive message after 5 seconds (demo mode).

Integration Code

<script>
  window.CHATWIDGET_CONFIG = {
    apiKey: 'YOUR_API_KEY',
    apiUrl: 'https://api.cloud-ru-ai.ru',
  };
</script>
<script src="https://cdn.cloud-ru-ai.ru/widget.js" async defer></script>
